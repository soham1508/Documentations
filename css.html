CSS Language Documentation

1. Introduction to CSS

CSS (Cascading Style Sheets) is a style sheet language used to describe the look and formatting of a document written in HTML or XML. It enables the separation of content from design.

2. CSS Syntax

CSS rules consist of selectors and declarations. Selectors target HTML elements, and declarations define styling properties.

selector {
    property: value;
}

3. CSS Selectors

CSS selectors specify the elements to which styles should apply. Some common selectors are:

  • Element Selector: targets HTML tags, e.g., p, h1.
  • Class Selector: targets elements with a class, e.g., .myClass.
  • ID Selector: targets elements with a specific ID, e.g., #myId.

4. Colors

CSS allows color specification using keywords, hex codes, RGB, RGBA, HSL, and HSLA values.

color: red;
background-color: #00FF00;

5. Backgrounds

The background properties define the background effects of an element, including color, image, position, and repeat.

6. Borders

Borders are lines surrounding an element. CSS provides border styling with properties such as border-width, border-style, and border-color.

7. Margins

Margins create space around elements, outside the border.

8. Padding

Padding creates space inside the element, between the content and the border.

9. Box Model

The CSS box model represents the structure of an element, including margin, border, padding, and content areas.

10. Text Properties

Text properties control the appearance of text, including color, font-size, text-align, and line-height.

11. Fonts

CSS allows you to specify font families, styles, weights, and sizes, as well as custom web fonts through @font-face.

12. Flexbox

Flexbox is a CSS layout model that arranges items in rows or columns and distributes space dynamically. It is useful for building flexible layouts.

13. CSS Grid

The CSS Grid layout is a two-dimensional system for creating complex, responsive layouts with rows and columns.

14. Positioning

CSS positioning properties (static, relative, absolute, fixed, and sticky) define how elements are positioned within the layout.

15. Transitions and Animations

CSS allows for smooth transitions between states, as well as animations to create dynamic effects on elements.

transition: background-color 0.5s ease;
@keyframes example {
    from { opacity: 0; }
    to { opacity: 1; }
}

16. Media Queries

Media queries enable responsive design by applying styles based on device properties such as width, height, and orientation.

@media (max-width: 600px) {
    body { font-size: 14px; }
}

Conclusion

CSS is essential for designing the look and layout of web pages, with a wide range of properties and tools that make web content visually engaging and responsive.
